 BUG-002 – Campo e-mail aceita formato inválido

 Ambiente
- Navegador: Google Chrome
- Sistema Operacional: Windows 10

 Pré-condição
Acesso à página de cadastro

 Passos para Reproduzir
1. Acessar a página de cadastro
2. Preencher o campo e-mail sem "@"
3. Preencher os demais campos corretamente
4. Clicar em "Cadastrar"

 Resultado Esperado
O sistema deve impedir o cadastro e exibir mensagem de e-mail inválido.

 Resultado Obtido
O sistema permite o cadastro com e-mail inválido.

 Severidade
Média

 Prioridade
Média
