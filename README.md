# Portfólio QA – Abimael de Sousa

Este repositório apresenta exemplos práticos de atividades realizadas por um
QA Júnior, com foco em testes manuais e boas práticas de qualidade de software.

##  Estrutura do Repositório

- *01-casos-de-teste**  
  Casos de teste manuais documentados para validação de funcionalidades.

- *02-relatorios-de-bug**  
  Relatórios de bugs com passos para reprodução, resultado esperado e obtido.

##  Objetivo

Demonstrar conhecimento prático em:
- Criação de casos de teste
- Identificação e documentação de bugs
- Organização de artefatos de QA
